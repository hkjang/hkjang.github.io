Why This Site

템플릿 소개 페이지가 아니라, 작업량과 실행력을 보여주는 전시관

오래된 정적 포트폴리오 대신, 지금 어떤 프로젝트를 만들고 있고 얼마나 꾸준히 업데이트하고 있는지 바로 확인할 수 있도록 설계했습니다.

01

실시간 신뢰도

GitHub API에서 최신 저장소 목록을 직접 가져와서 배포 이후에도 포트폴리오가 계속 살아 움직입니다.

02

대량 프로젝트 탐색

100개가 넘는 프로젝트도 검색, 언어 필터, 정렬로 빠르게 훑어볼 수 있게 구성했습니다.

03

홍보용 메시지 정리

결과물만 나열하지 않고, 어떤 스타일로 만들고 배포하는 사람인지까지 전달되도록 카피와 구조를 다듬었습니다.

Keep Iterating
